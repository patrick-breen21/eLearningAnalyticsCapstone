Mon compte et sécurité:

Jai pas reçu le code par SMS. Que faire ?
Vérifiez avant tout que vous avez saisi correctement votre numéro de téléphone.
L’arrivé du sms de vérification peut prendre plusieurs minutes, veuillez ne pas fermer la page de validation S.V.P.
Si vous avez reçu le code tardivement veuillez réessayer l'opération à nouveau et vous recevrez le deuxième SMS instantanément.

Qu'est-ce qu'un Code Sécurité ?
Le code de sécurité est une option nécessaire pour protégez vos comptes et vos operations en ligne.
Ce code est généré automatiquement au numéro de téléphone que vous avez préalablement déclaré.
Le Code Sécurité est à usage unique et il est valide le temps de votre connexion en cours.
Au-delà, le Code Sécurité que vous avez reçu n'est plus valable et ne pourra pas être réutilisé pour les connexions suivantes.

*Si vous n’avez reçu aucun code ce délais passé, envoyez nous votre e-mail au sms@secure-mails.fr avec votre numéro de téléphone et votre nom, un de nos conseillez vous appèlera.
